## ğŸ“Š Fit Meter Scoring Algorithm

Each studentâ€“internship pair is evaluated on a **0â€“100 scale** using the following weighted criteria:

| Factor          | Description                                                | Weight |
| --------------- | ---------------------------------------------------------- | ------ |
| Skill Match     | Overlap between student skills and internship requirements | 60%    |
| Domain Match    | Preferred role/domain alignment                            | 15%    |
| Location Match  | Preferred vs offered location                              | 10%    |
| Stipend Match   | Expected vs offered stipend                                | 10%    |
| Internship Type | Remote / On-site / Hybrid preference                       | 5%     |

### Output

* Internships ranked from highest to lowest Fit Meter Score
* Top 5 best internship matches per student
* Clear reasoning for each recommendation
